# Web API Tester

这是一个用于测试浏览器控制 API 的 Web 界面，已集成到浏览器控制服务器中。

## 使用方法

**只需启动浏览器控制服务器，测试界面会自动可用！**

```bash
npm install
npm run build
npm start
```

然后在浏览器中打开 `http://127.0.0.1:18791/` 即可使用测试界面。

## 功能特性

- ✅ 所有 API 接口的完整列表
- ✅ 每个接口的参数表单
- ✅ 实时发送请求
- ✅ 格式化显示响应结果
- ✅ 支持认证 Token
- ✅ 支持查询参数和请求体
- ✅ 一键复制响应结果
- ✅ 响应式设计，支持移动设备
- ✅ 自动检测服务器地址（无需手动配置）
- ✅ 无 CORS 问题（同源访问）

## 注意事项

1. 测试界面会自动使用当前页面的地址作为 API 服务器地址
2. 如果启用了认证，在 "Auth Token" 中输入 Bearer token
3. 所有 API 请求都会发送到同一个服务器，无需担心跨域问题
